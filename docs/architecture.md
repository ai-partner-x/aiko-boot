# AI-First Framework 架构说明

## 一、项目定位

**AI-First Framework** 是一个面向 AI 时代的 TypeScript 全栈开发框架，采用 **Spring Boot 风格** 的装饰器 API，让 AI 能够高效理解和生成代码。

## 二、技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端 | React 18 + Next.js 16 + shadcn/ui |
| 后端 | Next.js API Routes + Controller |
| 依赖注入 | tsyringe + 自定义装饰器 |
| 数据校验 | class-validator + react-hook-form |
| ORM | 自研（兼容 MyBatis-Plus / JPA） |
| 构建 | pnpm + TypeScript 5.9 + tsup |

## 六、核心装饰器 API

### Web 层 (@ai-first/nextjs)

```typescript
@RestController({ path: '/users' })
export class UserController {
  constructor(private userService: UserService) {}

  @GetMapping()
  async list(): Promise<User[]> { ... }

  @GetMapping('/:id')
  async getById(@PathVariable('id') id: string) { ... }

  @PostMapping()
  async create(@RequestBody() dto: CreateUserDto) { ... }

  @PutMapping('/:id')
  async update(@PathVariable('id') id: string, @RequestBody() dto: UpdateUserDto) { ... }

  @DeleteMapping('/:id')
  async delete(@PathVariable('id') id: string) { ... }
}
```

### 业务层 (@ai-first/core)

```typescript
@Service()
export class UserService {
  constructor(private userMapper: UserMapper) {}

  async getAllUsers(): Promise<User[]> {
    return this.userMapper.selectList();
  }
}
```

### 数据层 (@ai-first/orm)

```typescript
@Mapper(User)
export class UserMapper extends BaseMapper<User> {}

@Entity('t_user')
export class User {
  @TableId({ type: IdType.AUTO })
  id!: number;

  @TableField()
  name!: string;

  @TableField()
  email!: string;
}
```

### 依赖注入 (@ai-first/di)

```typescript
@Injectable()      // 标记可注入
@Singleton()       // 单例模式
@Scoped()          // 请求作用域
```

## 七、自动化机制

### postinstall 自动生成

安装 `@ai-first/nextjs` 后自动：

1. **扫描** `api/controller/*.controller.ts`
2. **扫描** `api/service/*.service.ts`
3. **扫描** `api/mapper/*.mapper.ts`
4. **生成** `app/api/[...path]/route.ts`

```
pnpm install → postinstall 脚本 → 自动生成路由文件
```

### 生成的 route.ts

```typescript
/**
 * Auto-generated by @ai-first/nextjs
 * DO NOT EDIT
 */
import 'reflect-metadata';
import { createApiRouter } from '@ai-first/nextjs';
import '@/api/mapper/user.mapper';
import '@/api/service/user.service';
import { UserController } from '@/api/controller/user.controller';

export const { GET, POST, PUT, DELETE, PATCH } = createApiRouter([UserController]);
```

## 八、开发工作流

```
1. 创建 Entity     →  api/entity/xxx.entity.ts
2. 创建 Mapper     →  api/mapper/xxx.mapper.ts
3. 创建 Service    →  api/service/xxx.service.ts
4. 创建 Controller →  api/controller/xxx.controller.ts
5. pnpm install    →  自动生成路由
6. pnpm dev        →  启动开发服务器
```

## 九、与 Spring Boot 对照

| Spring Boot | AI-First Framework |
|-------------|-------------------|
| `@RestController` | `@RestController` |
| `@GetMapping` | `@GetMapping` |
| `@PostMapping` | `@PostMapping` |
| `@PathVariable` | `@PathVariable` |
| `@RequestBody` | `@RequestBody` |
| `@Service` | `@Service` |
| `@Repository` | `@Mapper` |
| `@Entity` | `@Entity` |
| `@Autowired` | 构造函数注入 |
| `MyBatis BaseMapper` | `BaseMapper<T>` |

## 十、设计原则

1. **AI 友好** - 装饰器 API 易于 AI 理解和生成
2. **零配置** - postinstall 自动生成框架文件
3. **类型安全** - 全程 TypeScript 强类型
4. **Spring Boot 风格** - 降低 Java 开发者学习成本
5. **分层清晰** - Controller → Service → Mapper → Entity
